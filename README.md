# Noctalia plugins — mikopetryk

Third-party plugins for the [Noctalia](https://noctalia.dev) Quickshell-based shell.

## Install

In Noctalia: **Settings → Plugins → Sources → Add source**

```
https://github.com/mikopetryk/noctalia-plugins
```

Then switch to **Available**, find the plugin, and click **Install**.

## Plugins

| Plugin | Description |
| --- | --- |
| [`workspace-profiles`](workspace-profiles/) | Manage niri workspaces as profiles — named workspaces, pinned apps, all-screens workspaces, linked cross-monitor switching. |
